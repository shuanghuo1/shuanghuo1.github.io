Weather Clock | 齿轮天气钟
A fun experience of getting weather status.

用一种有趣的方式查询天气信息

The clock can report the real time weather data based on the online weather data from OpenWeatherMap API via Wifi. The first round of fast spinning is to demonstrate that different color lights correspond with different weather statuses. For the second round, gears spin slowly to point to the current weather status. One supply powers the whole setup.

天气钟能通过Wifi读取网络服务商提供的开放天气数据。齿轮第一圈快速旋转是用来展示不同颜色的灯光对应不同天气，而第二圈时齿轮指针则缓慢旋转直到指向代表当前天气状况的图标。只需插一个普通家电插座就能工作。

Brief Introduction of the Making Process

简略制作过程说明

The making process of this project contains many trials and errors. It was built up little by little. And there are still lots of things to work on if higher stability and accuracy is expected for a future version.

这个项目的制作过程中包含了很多的尝试与失败。它是一点一点搭建起来的。如果有机会进行下一个版本的改进，更高的稳定性和精确性将是目标，因此还有很多事情可以做。
